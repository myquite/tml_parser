What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages.

Key Concepts:

  • HTML describes the structure of a web page
  • HTML elements are represented by tags
  • Tags usually come in pairs: opening and closing

Basic Structure:

<!DOCTYPE html>
        <html>
          <head>
            <title>My Page</title>
          </head>
          <body>
            <h1>Hello World!</h1>
          </body>
        </html>

Common HTML Elements

Here are some essential HTML elements you'll use frequently:

  • Headings: <h1> through <h6>
  • Paragraphs: <p>
  • Links: <a href="...">
  • Images: <img src="..." alt="...">
  • Lists: <ul>, <ol>, <li>
  • Divisions: <div> and semantic elements like <header>, <nav>, <main>

Example: Personal Portfolio Page

Here's a complete HTML document for a personal portfolio:

<!DOCTYPE html>
          <html lang="en">
          <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Jane Doe - Web Developer</title>
          </head>
          <body>
            <header>
              <h1>Jane Doe</h1>
              <nav>
                <a href="#about">About</a>
                <a href="#projects">Projects</a>
                <a href="#contact">Contact</a>
              </nav>
            </header>
            <main>
              <section id="about">
                <h2>About Me</h2>
                <p>I'm a passionate web developer...</p>
              </section>
            </main>
            <footer>
              <p>&copy; 2024 Jane Doe</p>
            </footer>
          </body>
          </html>

Key Takeaways:

  • Notice the semantic HTML5 elements (<header>, <nav>, <main>, <section>, <footer>)
  • All content is properly nested
  • The document has a clear structure
