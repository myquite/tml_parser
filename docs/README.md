## üí° Tips

1. **Start with examples**: The `examples/` directory contains working TML files you can use as templates
2. **Validate early**: Use XSD validation to catch errors before generating HTML
3. **Customize templates**: The template system is flexible - modify templates to match your brand
4. **Check the schema**: When in doubt, refer to `tml-v0.1.xsd` for the authoritative structure
